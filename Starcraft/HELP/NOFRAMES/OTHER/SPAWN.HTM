SPAWN

To install a Spawn copy:

  • Insert the Starcraft disc into your CD-ROM drive.
  • The StarCraft splash screen Should appear.
    • If the splash screen does not appear:
    • Double click on the "My Computer" icon
    • Double click on your CD-ROM icon
    • Then double click on the install or Setup icons.
  • Click on Install Spawn.
  • Review the License Agreement and click Agree to continue.
  • Enter your name and the CD Registration Key located with the game documentation. Click OK to proceed.
  • Choose the directory where you would like Starcraft to be installed. By default, Starcraft will be installed in C:\Program Files\Starcraft. Click OK to begin the installation.
  • A dialog box will appear once Starcraft has finished installing successfully. Click OK to continue.
  • The main menu will be displayed again. Click on Play Starcraft Spawn to begin the game.
  • For additional information, please check the Starcraft manual and the online help files.

A spawned player can:

  • Join any multi-player games created by the person who spawned him.
  • Use the map editor

A spawned player cannot:

  • Play single player games
  • Create a multi-player game
  • Play in any ladder games

Notice that even spawned players will have the ability to use the map editor.
