HOW TO E-MAIL BLIZZARD
To help our technical support representatives assist you as quickly as possible, please include the following information when sending e-mail to support@blizzard.com

Personal information
  • Name
  • E-mail address
  • Phone number
Information on the problem
  • What game is it?
  • Type of problem (sound, video, error message, ...)
  • Is this the first time the problem has happened?
  • What is the error message? (if you received one)
  • If the error occurs consistantly, are there steps you can take to recreate it?
Computer information
  • Computer manufacturer
  • Video and Sound card
  • Amount of RAM
  • CPU speed
  • CD-ROM (Speed and manufacturer)
