BATTLE.NET ERRORS

General Troubleshooting

  • Check with your ISP and/or network admin to make sure your network is not restricting outgoing packets on Ports 116 & 118 of your router. Also make sure port 6112 is open for UDP packets.
  • If you have an Aptiva with an MWave card, you need the MWave update

Battle.net was unable to properly identify your application version.

  • You'll need to uninstall & reinstall Starcraft. This is most often caused by disk errors.

The update has been interrupted. No changes were made. (During patch download)

  • (Note: At the time of writing this support page there are no patches. The error message is a Battle.net error message that will be displayed if there is a problem downloading a patch.)
  • Download the Starcraft patch and install it. If the patch fails to install correctly, uninstall & reinstall Starcraft.

Unable to connect to Battle.net.

  • See the steps outlined above in General Troubleshooting.

Your Internet connection is either very poor or is not processing UDP packets.

  • See the steps outlined above in General Troubleshooting.

Your system has not been configured to support 32-bit Internet applications.

  • Try connecting to your ISP before you start up Starcraft. If this still doesn't work, contact your ISP about obtaining a 32-bit connection.
