INSTALLING AND UNINSTALLING STARCRAFT

To Install:

  • Insert the Starcraft CD
    • Select Install Starcraft
  • If the auto play screen does not pop up
    • double-click on the "My Computer" icon
    • Double-click on the CD icon
    • Double-click on the Setup icon
    • The install program will then give you directions on how to install.

To Uninstall:

  • Click on "Start"
  • Highlight "Programs"
  • Highlight "StarCraft"
  • Click on "Uninstall"
