DIAL UP NETWORKING

This error means that you do not have a 32bit Winsock compliant Internet connection.
You will need to contact your Internet Service Provider for help in configuring your system to use Windows Dial Up Networking.

CONNECTING TO BATTLE.NET WITH AOL

  • You must have a 32-bit Winsock connection. If you're connecting through AOL, you need to make sure you're using AOL 3.0 for Windows 95 (32-bit version, 11.4 mb) or a newer version.
  • Once you have installed this version of AOL you will want to do the following to connect to Battle.net:
    • Connect to AOL
    • Click on the world icon in the center of your tool bar
    • Minimize AOL
    • Start Starcraft
    • Select Multiplayer
    • Create/Select your character
    • Choose Battle.net from the list of connection methods
  • If you are still having problems with the connection try dialing into AOL useing a different AOL dial-up phone number.
