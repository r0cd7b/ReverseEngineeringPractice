WINDOWS UPDATES

Microsoft Security Updates
Search for both: "Out-of-Band" and "Invalid ICMP Datagram Fragments".
Also check Microsoft's web site at www.microsoft.com for any new security fixes.

To determine the version of Windows 95 you are using:

  • Click on "Start"
  • Click on "Settings"
  • Click on "Control Panel"
  • Double click on the "System" icon
  • The version should be listed under Microsoft Windows 95.

For Windows95, we recommend the following Windows updates from Microsoft:

  • Windows version 4.00.950 only:
    • Win95 service pack.
  • Windows version 4.00.950 and 4.00.950a only:
    • Kernel32 Update. This may help systems that encounter errors in Kernel32.
    • Password List Update. This fixes a bug which may cause problems with dial-up networking.
  • All Windows95 versions:
    • Click here to check your CD-ROM and Hard disk settings
    • Regclean. This program fixes some bugs associated with the Windows Registry.


Windows NT
  • Users should make sure they have the most recent Service Pack for NT installed.
  • We also recommend installing any of the Microsoft Hot Fixes that apply to your computer.
  • Make sure to install all the security updates that apply to your computer to help prevent remote users from crashing your system over an Internet connection.
